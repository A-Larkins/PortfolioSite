This was my term project for my class called Server Side Web Application Development. It is a website for studying, where users can create, edit, and study flashcards. It is a Client-Server system with Multi-Tier architecture that includes a complete login system with user registration and security functions. It was made using the .NET framework and used ASMX SOAP and ASP.NET Core REST web APIs that used serialization to send and receive form data in JSON and XML. I utilized stored procedures in T-SQL to perform database functions. For the UI, I used bootstrap and made custom user controls using ASCX and used a few different dynamic data displays. The source code is available on: <a href="https://github.com/A-Larkins/QuizFlash" target="_blank">GitHub</a>

The 52 Week Movie Challenge is a logbook — each week you watch a film matching a themed prompt and record the details. Filling it in meant bouncing between IMDb and Google for every entry, so I wrote something to do it for me.

Type a title and it queries OMDb and TMDB, merges the results, and returns everything the logbook asks for: year, runtime, aspect ratio, genre, rating, cast, director, producer, writer, composer, and awards. For the fields the APIs don't cover cleanly it opens a targeted Google search and the IMDb quotes page so I can grab the rest.

Started as a console app published to my system path, so logging a film was one command. It's now a shared core library with both a console front end and an Avalonia desktop app on top.

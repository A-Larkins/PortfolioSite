The 52 Week Movie Challenge is a movie logbook organized into themed categories — each week you pick a film matching a prompt, watch it, and log detailed information about it. I started it because I love movies, and quickly realized I was spending far too long clicking around IMDb and Google gathering the same fields for every entry.

So I wrote a helper. It queries OMDb and TMDB, combines the results, and outputs every detail the logbook asks for in one place. For the things that aren't cleanly available from an API — aspect ratio, certain awards details — it opens targeted browser searches and IMDb quote pages so I can grab the rest quickly.

It started as a console app, published as a self-contained executable and added to my system path, so logging a film was a single terminal command instead of a dozen browser tabs. I've since split it into a core library with an Avalonia desktop UI on top, so it runs as a proper macOS app that remembers its window placement. The console version still works — the shared core means both front ends do the same thing.
